RUNNING

Install dependencies - Xampp and $ yarn 

Run Development servers:

Create database (db_techtest) in mysql

** Running Yarn **

$ yarn install

$ yarn run dev

$ yarn start

Once the server is running, you can visit http://localhost:3000/
